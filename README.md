# ChestAI — Next-Gen Chest X-Ray Diagnostics

An AI-powered web application for chest X-ray analysis built using DenseNet121.
Upload a chest X-ray and get instant probability scores for 5 conditions along with a GradCAM heatmap showing which regions the model focused on.

## Conditions Detected

* Pneumonia
* Infiltration
* Effusion
* Cardiomegaly
* Atelectasis

## Features

* Deep Learning model (DenseNet121) trained on NIH Chest X-Ray dataset
* GradCAM visualization for model explainability
* Adjustable detection threshold
* Clean, modern UI

## How to Use

1. Open the website
2. Upload a chest X-ray image
3. Click Analyze X-Ray
4. View probabilities and heatmap
